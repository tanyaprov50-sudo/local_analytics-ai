После создания репозитория на GitHub выполни эти команды:

# Замени YOUR_USERNAME на твой GitHub username
git remote add origin https://github.com/YOUR_USERNAME/local-analytics-ai.git

# Переименуй ветку в main
git branch -M main

# Загрузи код на GitHub
git push -u origin main

Пример:
git remote add origin https://github.com/tanyaprov50-sudo/local_analytics-ai.git
git branch -M main
git push -u origin main
